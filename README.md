# ✨ Hyprland Config

A clean, aesthetic and minimal **Hyprland window manager configuration** designed to decorate and enhance your Wayland desktop experience.

This setup focuses on:

- Smooth animations  
- Modern blur & rounded windows  
- Clean tiling workflow  
- Functional dashboard & launcher  
- Lock screen & idle management  
- Lightweight but visually pleasing design  

Turn your Linux desktop into a beautiful, fast and productive tiling environment.
